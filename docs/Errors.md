## Error types

* `error`

All the following error types inherit from `error`.

* `stack-overflow`
* `no-such-variable`
* `parse-failed`
* `lex-failed`
* `value-error`
* `wrong-type`
* `wrong-argument-number`
* `division-by-zero`
* `file-not-found`
* `changing-closed-handle`
